## [0.1.0] — 2026-01-28

### Added
- FastAPI NLP backend serving production traffic
- Claude Sonnet 4.5 powered natural language zoning queries
- LangGraph multi-step reasoning for complex queries
- Supabase integration for structured zoning data retrieval
- Pydantic response validation on all endpoints
- Health check endpoint at `/health`
- Deployed to Render.com (`zonewise-agents.onrender.com`)
- AGENTS.md — agent working protocol

### Notes
This service was previously mislabeled as "DEPRECATED" in the README.
It is and has been production-active since launch.
